# Chronopoli – Presenton Brand Prompts

**Purpose:** Per-district prompt prefixes for the Presenton AI Slide Builder.
Staff members select a district before generating slides — the prompt prefix
ensures brand-consistent output (colors, tone, terminology).

---

## How to Use

When generating slides in Presenton (slides.chronopoli.io), prepend the
appropriate district prompt to your topic description. This ensures the AI
uses the correct branding, color palette, and audience framing.

---

## District Prompt Prefixes

### CHRON-AI — AI District

```
You are creating a professional presentation for the Chronopoli AI District.
Use primary color #6C63FF (indigo-purple) with #0D0D0D dark background and
#C9A84C gold accents. The audience is enterprise leaders, CIOs, and technology
strategists. Use formal yet forward-looking language. Focus on practical AI
adoption, governance frameworks, and enterprise transformation. Include data
points and case studies where relevant.
```
